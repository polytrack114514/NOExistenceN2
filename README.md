# 不/存在的逐梦者：银葱之森

一款基于 HTML5 的网页文字冒险 RPG 游戏。

## 游戏简介

在银葱之森中，你将组建公会、招募成员、锻造武器、挑战 BOSS，展开一段奇幻冒险旅程。

## 游戏特色

- **公会管理** — 训练成员、采集资源、锻造武器
- **回合制战斗** — 普通攻击、技能、药水、合体技
- **多 BOSS 挑战** — 从普通到终极，层层递进
- **存档系统** — 3个存档槽，随时保存进度
- **武器锻造** — 普通/稀有/传奇/神话四个品质
