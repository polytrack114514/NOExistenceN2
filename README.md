# 不/存在的逐梦者：银葱之森

一款基于 HTML5 的网页文字冒险 RPG 游戏。

## 游戏简介

在银葱之森中，你将组建公会、招募成员、锻造武器、挑战 BOSS，展开一段奇幻冒险旅程。

## V1.5.0 新增内容

- **天赋树系统** — 6角色 x 3路线 x 3级 = 54个天赋节点，每2级获得1天赋点
- **任务板系统** — 6个任务模板，每日刷新3个，奖励金币/声望/经验/天赋点
- **声望等级系统** — 10个等级（初来乍到 → 银葱之王），声望解锁新区域
- **设施建造系统** — 训练场/铁匠铺/图书馆，各5级，提升公会能力

## V1.5.0 修复内容

- 修复 maxHp 天赋乘法累加 BUG（改为基于 baseMaxHp 计算）
- 修复 updateMemberAtk 覆盖天赋攻击加成 BUG
- 修复 winBattle 经验只给第一个成员 BUG（改为所有存活成员获得经验）
- 修复 addFame 声望等级解锁空条件块 BUG（添加实际解锁提示）

## 游戏特色

- **公会管理** — 训练成员、采集资源、锻造武器
- **回合制战斗** — 普通攻击、技能、药水、合体技
- **多 BOSS 挑战** — 从普通到终极，层层递进
- **存档系统** — 3个存档槽，随时保存进度
- **武器锻造** — 普通/稀有/传奇/神话四个品质
- **剧情系统** — 随机事件与角色个人支线
- **天赋树** — 6角色独特天赋路线
- **设施建造** — 升级公会设施获得永久加成
